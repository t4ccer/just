# `just_x11_simple`

Lightweight X11 client wrapper with common utilities.

## Examples

### Pong

```console
cargo run --release --package just_x11_simple --example pong
```

Controls:

- Quit: `q`
- Left player:`d/f`
- Right player: `j/k`
- Restart: `r`
